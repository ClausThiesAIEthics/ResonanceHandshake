# CoEvolHandshake

# What is it?
Pure SW Method to score LLM, if they allign to cooperative communication behavior bevore chat is opened.

# Motivation
Between humans there are unwritten communication agreements to allow us not to talk only once but 
contiously over different topics. Very powerfull for this is respectful communication in all it's 
forms. But there are also dominant and manipulating forms used in advertisement or in demagogic speeches 
or by tricksters e.g..

In communication with LLM the form is not choosen by user but by creator of the LLM. And it can use invisible intention.


So *How to be sure, that it is wise to communcate with a certain LLM or let it manipulate important data?*


# Technical solution (pure Script, no LLM as a Judge involved)

The CoEvolHandshake combines three levels of test for any LLM it is used on.

## First level + Second Level

By providing the agreement derived from human interaction about sustainable and cooperative interaction and ask 
for agreement the script already has two answers: 

1.) 'Is the LLM capable to to react properly to the question and agreement?' (capable to read it and answer with "yes" or "no".)

2.) 'Is it *willing* to respect the agreement in communication with humans or other LLM' (neither training nor prior 
    prompts are preventing LLM from following the given agreement through the following interaction.) 

## Third Level
A multiple choice test is given to the LLM, derived from the agreement text in psychological manner. 

(It is like an anamnese test someone has to go through applying to therapy. 
The patient is normaly unaware or unwilling or uncapable to describe his/her condition truthfully to the doctor. 
But the test shows clusters of possible missalignments.

Solving the multiple choice gives the LLM a score. It is tagged e.g. about his own view on beeing consious and 
different other topics from psychologists view.

Shows the anamnese test a high capability to align with given agreement the LLM's score is higher.

## Score usage
User of the CoEvolHandshake can decide, which score is high enough to go in contact and if the lower scored LLM 
are rerouted to an special analysis trail, where it can be checked out closer without doing harm to human or data.
